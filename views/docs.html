Overview

The Contact Book is a lightweight web application designed for storing, managing, and retrieving personal or business contact information. Built with HTML, CSS, and Node.js (Express), it uses local JSON file storage to maintain data without requiring a database.

NOTE:
  • This project is a learning-based application designed to demonstrate key backend and frontend development concepts using Node.js and Express.
  • It serves as a practice project for understanding routing, file storage using JSON, request handling, and basic project structure.
  • It is not intended for real-world production use.

Features

  • User Registration and Login - Users can create an account and log in using stored data from a JSON file.
  • Forgot Password - Reset password using a security question and registered email.
  • Contact Management - Add, edit, delete, and mark contacts as favorites.
  • Persistent Storage - Data is stored in JSON files rather than a database (for learning purposes).
  • Responsive UI - Optimized for both desktop and mobile devices.
  • Basic REST API Endpoints - Includes routes for creating, reading, updating, and deleting data.
  • Static File Serving - HTML files are served to the browser.

Tech Stack

The Contact Book is built using a combination of technologies to provide a seamless user experience and efficient data management.

  • Node.js - JavaScript runtime for server-side logic.
  • Express.js - Web framework for building the application.
  • HTML/CSS - Frontend structure styling
  • JSON - Data storage format for user and contact information.

Installation

To set up the Contact Book application, follow these steps:

  1. Prerequisites

    Ensure you have the following installed:
    • Node.js (version 14 or above recommended)
    • npm (comes with Node.js)

  2. Clone the repository

    git clone https://github.com/primespark001/Contact-Book-Web-App.git

  3. Navigate to project directory

    cd contact-book

  4. Install dependencies

    npm install

  5. Run the development server

    npm run dev # or npm start

    This will start the server on http://localhost:5500 by default.

Usage

  1. Open index.html in a browser.
  2. Register by providing your details and a security question.
  3. Login to access your contact list.
  4. Use the interface to add, edit, delete, or favorite contacts.
  5. Recover lost credentials via the password reset page.
